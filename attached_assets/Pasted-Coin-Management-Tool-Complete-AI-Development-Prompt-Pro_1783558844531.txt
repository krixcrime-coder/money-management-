Coin Management Tool – Complete AI Development Prompt

Project Name

Coin Management Tool

Purpose

Build a web application that helps users manage their coins with discipline while playing a coin-based game.

Important:
This tool does not predict any game result, does not hack the game, does not analyze the game's algorithm, and does not guarantee profit.

Its only purpose is to help users follow a daily money management strategy, record progress, manage targets, and maintain discipline.

---

About the Game

The game works like this:

- Every new user receives 100 coins after creating an account.
- The goal is to reach 10,000 coins within one month (30 days).
- Players can play anytime during the month.
- There is no daily play limit.
- The reward is given only if the player has 10,000 coins or more at the end of the month.
- The game contains:
  - Fast Button
  - Slow/Second Button
  - Random Result Chart

Example:

Player bets 10 coins on Fast.

If Fast wins:
+10 coins

If Fast loses:
-10 coins

The game result is completely random.

The Coin Management Tool must NEVER claim that it knows or predicts future outcomes.

---

Purpose of This Tool

This tool only helps users by:

- Managing daily targets.
- Managing daily stop-loss.
- Preventing overplaying.
- Recording progress.
- Tracking recovery after losses.
- Saving user history.
- Unlocking new strategy each day.
- Keeping users disciplined.
